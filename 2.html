What is CSS?

As mentioned earlier, CSS is the language that specifies how documents are displayed to users — style, layout, and so on.

CSS can be used for styling very basic text documents — for example, changing the color and size of titles and links. Also, it can be used to create layouts — for example, converting a single text column (en-US) (a layout with sidebar for the main content area and related information).
